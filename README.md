# adsense

광고를 내 블로그에 달아서 수익을 창출 할수 있는 방법이다

카카오도 있고 여러가지 있지만 광고금액을 측정할때 구글만 CPM(Cost per mille:웹 페이지에서 광고가 1,000번 노출됐을 때의 비용) 노출만으로 과금을 해주는 유일한 곳이다.

**선행조건** 
- 블로깅 관련 컨텐츠가 많이? 있어야 된다.. 그러므로 성공하는 사람이 나오면 후속 클래스를 열겠습니다.
